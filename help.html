Справка по текстовому редактору

Меню

  • Файл
    • Создать: Создает новый файл для редактирования.
    • Открыть: Открывает существующий файл для редактирования.
    • Сохранить: Сохраняет текущий текст в файле. Если файл уже существует, он перезаписывается.
    • Сохранить как: Сохраняет текущий текст в новом файле.
  • Правка
    • Отмена (Undo): Отменяет последнее действие (текстовое изменение).
    • Повтор (Redo): Возвращает отмененное действие.
    • Очистить: Очищает содержимое текстового редактора.
  • Действия
    • Обработка регулярных выражений: Подробности о регулярных выражениях.
    • Детерминированный конечный автомат: Подробности о конечных автоматах.

Горячие клавиши

  • Ctrl + Z: Отменить (Undo).
  • Ctrl + X: Вырезать выделенный текст.
  • Ctrl + C: Скопировать выделенный текст.
  • Ctrl + V: Вставить текст из буфера обмена.
  • Ctrl + A: Выделить весь текст.
