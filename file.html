Справка по текстовому редактору

Файл

  • Создать: Создает новый файл для редактирования.
  • Открыть: Открывает существующий файл для редактирования.
  • Сохранить: Сохраняет текущий текст в файле. Если файл уже существует, он перезаписывается.
  • Сохранить как: Сохраняет текущий текст в новом файле.
